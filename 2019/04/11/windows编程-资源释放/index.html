发表于 2019-04-11 | 分类于 Windows黑客编程

函数介绍

FindResource函数

  • hModule[in]:处理包含资源的可执行文件模块。若为NULL，系统从当前进程的模块中装载资源。
  • lpname[in]:指定资源名称。
  • ipType[in]:指定资源类型。

SezeofResource函数

  • hModule[in]:包含资源的可执行文件模块。若为NULL，系统从当前进程的模块中装载资源。
  • hResInfo[in]:资源句柄。必须由函数FindResource或者FindResourceEx来创建。

LoadResource函数

  • hModule[in]:包含资源的可执行文件模块。若为NULL，系统从当前进程的模块中装载资源。
  • hResInfo[in]:资源句柄。必须由函数FindResource或者FindResourceEx来创建。

LockResource函数

  • hResData[in]:装载资源的句柄。由LoadResource返回此句柄。
